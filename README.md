# Project Game

Онлайн-платформа для игр в компании друзей.

В будущем приложение будет содержать каталог разных party games: Мафия, Бункер,
Шпион, Кто я? и другие. На первом этапе реализована игра "Мафия": пользователи
могут создать комнату, пригласить друзей по ссылке, получить роли, играть по
фазам ночи и дня, голосовать и автоматически определять победителя.

## Стек

- Next.js
- TypeScript
- React
- Tailwind CSS
- Socket.IO
- In-memory storage для MVP

## Запуск

```bash
pnpm install
pnpm dev
```

Приложение откроется на `http://localhost:3000`.
